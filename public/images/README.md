# Image Placeholders

This directory contains sample images for the Instagram-style feed.

## Required Images:

### Profile Pictures (Stories)
- sample1.jpg - sample5.jpg (64x64px minimum, square format)

### Post Images  
- post1.jpg - post4.jpg (1080x1080px recommended, square format)

## Quick Setup with Placeholder Images:

You can use any square images or use placeholder services like:
- https://placehold.co/1080x1080
- https://picsum.photos/1080/1080

Or simply add your own images with these filenames:
- sample1.jpg, sample2.jpg, sample3.jpg, sample4.jpg, sample5.jpg
- post1.jpg, post2.jpg, post3.jpg, post4.jpg

The app will work fine without images - they will show placeholder avatars instead.
